Flooring-Mastery-Project
Flooring Mastery Project is a console based application built upon Eclipse IDE. It uses the basic principles of Java i.e Object Oriented Concept.This project is all about the flooring program of a SWC Corporation. When the application is run, it provides user the menu option to display orders, add order, edit an order, remove order, save the order in a text file or exit the program. This aplication is developed in such a way that if a user wishes to save an order, that order is saved with a file name of current date so that each file is created with specific date. Similarly, the order is displayed in according to the date given by the user.
